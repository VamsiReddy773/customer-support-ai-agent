
# Customer Support AI Agent

## Overview

This project is an AI-powered customer support chatbot that answers FAQ questions using Retrieval-Augmented Generation (RAG) and retrieves order details from a CSV dataset.

## Features

- FAQ answering using RAG
- Order lookup from CSV
- Streamlit web interface
- FAISS vector database
- HuggingFace embeddings

## Technologies

- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
